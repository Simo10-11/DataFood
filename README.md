# DataFood 🍔

Applicazione backend per la gestione e visualizzazione di prodotti alimentari (e-commerce) tramite API REST e database MySQL.

---

## 🚀 Features

* Connessione a database MySQL
* API REST 
* Recupero dati alimentari
* Servizio backend semplice e scalabile
* Frontend semplice ma efficace

---

## 🛠️ Tecnologie

* Angular
* Springboot
* MySQL
